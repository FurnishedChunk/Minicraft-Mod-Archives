# Minicraft God Mode 
  
God Mode has two different Mod. 

- About God Mode developed by Dezlad. 

>

- About God Mode developed by Andrew Canis. 

>In this mod of Minicraft you are god:
>1) You don't lose health
>2) You don't lose stamina
>3) You radiate light in the caves
>4) You have all the gem tools
